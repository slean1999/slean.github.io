动手搭建blog

1.在电脑中某个盘下一个文件夹用于存放blog数据(比如我建在D:\MyBlog)，在此文件夹(d:\MyBlog)下右键Git Bash打开命令行。执行

$ hexo s
```

顺利的话，Hexo Server已经启动了，在浏览器中打开 http://localhost:4000/，这时可以看到Hexo已为你生成了一篇blog。

2.GitHub下新建一个项目

步骤如下图：

Respository name 必须填yourblog.github.io，yourblog随便填，后面的不能改。在settings下Source选择一个分支。

3. 配置_config.yml

部署到Github前需要配置_config.yml文件，找到下面的内容对应修改为你的。

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
  type: git
  repo: https://github.com/yourname/xxxxx.github.io.git
  branch: master

# URL
## If your site is put in a subdirectory, set url as 'http://yoursite.com/child' and root as '/child/'
url: https://yourname.github.io/xxxxx.github.io/
root: /xxxxx.github.io
permalink: :year/:month/:day/:title/
permalink_defaults:

4.部署

hexo clean
hexo g -d

顺利的话，打开 https://yourname.github.io/xxxxx.github.io/应该可以看到你blog了。

其它

1.新建自己的blog

执行命令,则创建了新的。

$ hexo new "我的新文章"

2.makedown及编辑工具

打开文件夹下/soruce/_post/下有.md文件。找个makedown工具编辑你的文章吧。makedown语法

3.主题

git上有许多主题。我用的是next主题。在/themes/执行命令:

git clone https://github.com/iissnan/hexo-theme-next themes/next

可以看到themes文件夹下多了一个next文件夹。修改刚才的配置文件找到并修改:

# Extensions
## Plugins: https://hexo.io/plugins/
## Themes: https://hexo.io/themes/
theme: next

执行上面的部署操作，就可以了。
